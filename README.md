## Laravel Set up

**Follow these steps to set up the project:**
1. Open the **laravel_restapi** folder in **Visual Studio Code**
2. Open and start **Apache and MySQL** in your **XAMPP Control Panel**
3. Click the **admin** button in MySQL
4. Create your database.
5. In your terminal enter **cd laravel_restapi**
6. Update the **.env** file with your database credentials
7. Open terminal and run database migrations: **php artisan migrate**
8. Serve the Application: **php artisan serve**
9. The API will be accessible at **_http://127.0.0.1:8000_**.
10. In URL enter this link **http://127.0.0.1:8000/api/test**
11. If it displays "**Your REST API is Ready.**" you can set up the NextJs.

## NextJS Set Up
**A guide to setting up and running your Next.js project.**
1. Open terminal and run **npm run dev**
2. The application will be accessible at **_http://localhost:3000_**.
